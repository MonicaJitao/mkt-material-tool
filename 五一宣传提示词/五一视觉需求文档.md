# 五一劳动节营销视觉需求文档

## 项目名称
**《造城者》** 五一劳动节城市定制化营销活动

---

## 一、项目背景

### 1.1 时间节点
- **五一劳动节**：2026年5月1日
- **黄金周**：5月1日—5日

### 1.2 核心洞察
客群以制造业、科学研究和技术服务业、信息传输/软件/技术服务业企业主为主。
五一不只是假期，是对创造者的致敬。每一座城市的繁荣，都由劳动者的双手与智识构建。
让每位企业主感受到：你的行业，是这座城市的脊梁。

---

## 二、创意策略

### 2.1 主题
> **造城者**

### 2.2 核心概念
城市是被人造出来的。工厂、代码、实验室——不同的劳动形态，共同构成一座城市的骨骼。
五一，我们不说泛泛的"劳动光荣"，我们说：**这座城市，因你而立。**

背景图直接承载"城市 × 产业"双重信息——城市地标作为空间骨架，产业氛围渗透进光线、材质与前景细节。
制造业老板和软件公司老板收到同一张海报，都感觉说的是自己。

### 2.3 情感诉求
| 层面 | 表达 |
|------|------|
| 理性层 | 行业认同、产业自豪、经营价值 |
| 感性层 | 劳动光荣、创造者致敬、城市归属 |
| 归属层 | 我的行业，是这座城市的脊梁 |

## 三、视觉需求

### 3.1 设计风格

**明亮摄影融合风格**：AI 生成的城市×产业融合摄影，以晴天日光或黄金时刻为主基调，色彩鲜艳饱满，传递热情积极、充满活力的劳动节氛围。

| 参照来源 | 提取语言 |
|---------|---------|
| 商业城市摄影 | 高饱和度、明亮通透、城市生命力 |
| 节日庆典视觉 | 热烈色彩、积极向上、喜庆而不俗气 |
| 工业人文摄影 | 劳动者的自豪感、产业的蓬勃气象、有温度有故事 |
| 旅游城市宣传片 | 城市最美时刻、地标辨识度高、让人产生归属感 |

**核心视觉逻辑**：
- 背景图 = 城市地标场景 + 产业氛围的无缝融合（非拼贴，是一张完整的融合图）
- 时间设定为**晴天上午或黄金时刻（日落前1小时）**，光线充足、色彩饱满
- 城市感通过地标建筑、城市天际线、标志性自然景观传达
- 产业感通过前景材质、色彩基调、局部产业元素渗透
- 整体明亮、鲜艳、有活力，传递"劳动创造美好"的积极情绪

### 3.2 色彩体系

| 色彩 | 用途 | 说明 |
|------|------|------|
| 明亮天空/暖阳 | 全局背景基调 | 蓝天白云或暖金夕阳，高亮度高饱和度 |
| 城市专属色 | 主视觉色彩倾向 | 饱和度提升至鲜艳版本，作为图片主色调 |
| 劳动热情红 `#FF4D2E` | 全局强调色 | 用于标题色块、节日标识底色，热烈醒目 |
| 深色文字 `#1A1A1A` | 主文字（浅色背景区） | 城市名、文案、落款 |
| 纯白 `#FFFFFF` | 辅助文字 | 叠压在深色图像区域时使用 |

> 城市专属色在此版本中统一提升饱和度，以鲜艳明亮为准，避免暗沉。

### 3.3 视觉语言规范

**背景图生成方向**
- 时间：晴天上午10点光线 或 黄金时刻（日落前暖橙光）
- 城市地标清晰可辨，色彩鲜艳，天空湛蓝或暖橙
- 产业元素融入前景：以真实的产业场景局部为主（工厂外景、港口、科技园区等），而非材质纹理叠加
- 整体明亮通透，高饱和度，有蓬勃向上的生命力
- 禁止：暗调、压抑感、过度滤镜、卡通化、拼贴感

**构图**
- 开阔大气，城市地标居于画面中上部，天空占比不低于 30%
- 文字区域在下 1/4，背景图该区域适当压亮或留出干净色块
- 整体构图饱满，不追求"失重感"，转而追求"蓬勃感"

**色彩处理**
- 背景图整体饱和度 +15%—+25%，亮度适当提升
- 城市专属色作为图片主色调倾向，通过色调映射实现
- 文字区域底部可叠加城市专属色的半透明色块（透明度 60%—70%），确保文字可读性同时保留色彩活力

**留白**
- 天空区域作为视觉呼吸空间，不堆砌元素
- 文字区域简洁，信息层级清晰

### 3.4 字体规范

| 用途 | 字体 | 字重 | 字间距 |
|------|------|------|------|
| 城市名 | 思源黑体 / Noto Sans SC | Black 900 | +150 |
| 主标题/活动名 | 思源黑体 / Noto Sans SC | Bold 700 | +100 |
| 正文/文案 | 思源宋体 / Noto Serif SC | Regular 400 | +60 |
| 数字/英文 | Helvetica Neue | Bold | +30 |
| 落款/经理姓名 | 思源宋体 / Noto Serif SC | Regular 400 | +150 |

> 字体从宋体改为黑体为主，更有力量感和现代感，与热情积极的整体调性匹配。
