# 节日祝福海报工作流 · 标准规范

## 命名规范
```
HTML 文件：{城市}-{客户经理姓名}.html
图片文件：{城市}-{客户经理姓名}.png
```

---

## 一、底图生成（AI 图像）

**工具**：`mcp__tuzi-tools__generate_image`

**固定参数**：
- `quality`: `2k`
- `size`: `9x16`

**Prompt 模板**（节日主题可替换）：
```
Beijing Labor Day illustration 9:16 vertical. Center: large golden 3D "51" numerals.
Background: Great Wall across ink-wash misty mountains. Red silk ribbons wrap around the "51".
Foreground: construction worker, farmer, doctor in dynamic poses.
Sky: sunrise golden rays, white doves flying. Ground: gear and wheat ear motifs.
Colors: vermillion red, gold, celadon blue, white negative space.
Modern Chinese illustration style with traditional cloud patterns.
Clean layout, solid color at bottom.
Absolutely NO text, NO Chinese characters, NO words anywhere in the image.
```

> 关键约束：末尾必须加 `NO text, NO Chinese characters` 确保底图无文字。

---

## 二、HTML 尺寸规范

| 属性 | 数值 |
|------|------|
| 卡片宽度 | 450px |
| 底图高度 | 800px（9:16） |
| 文案区高度 | ~200px |
| 整体比例 | 9:20（450 × 1000px） |
| 渲染参数 | width=450, height=1000 |

---

## 三、字体规范

| 元素 | 字号 | 字间距 | 对齐 |
|------|------|--------|------|
| 主标语 | 18px | 0.16em | 左对齐 |
| 副标语 | 18px | 0.12em | 左对齐 |
| 城市高亮 | 29px | — | 继承 |
| 落款 | 18px | 0.2em | 左对齐 |

字体栈：`"Songti SC", "STSong", "SimSun", serif`

---

## 四、色彩规范

| 用途 | 色值 |
|------|------|
| 背景色 | `#0e0600` |
| 正文色 | `#f5e6b8`（暖白） |
| 城市高亮 | `#e8c84a`（鎏金） |
| 分割线 | `rgba(200,168,75,0.55)` |
| 渐变过渡 | `transparent → #0e0600` |

---

## 五、文案结构模板

```
主标语：每一项研究，都是对未来的承诺
副标语：劳动光荣，致敬每一个在【{城市}】仰望与探索的人
落款：  {公司}{客户经理}祝您节日快乐
```

---

## 六、批量生产步骤

1. **生成底图** → 保存图片 URL
2. **复制 HTML 模板** → 替换 `城市名` 和 `客户经理姓名`
3. **渲染图片** → `html_to_image`（width=450, height=1000）
4. **命名保存** → `{城市}-{姓名}.html` + `{城市}-{姓名}.png`
